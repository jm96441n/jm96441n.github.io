Cheat Sheet!

This Cheat sheet will be referencing Ruby Enumerables, and methods of Enumeration

.each {|x| do something to x} This method of enumeration loops through each element of an array or hash, where x is the element, and then what you want to perform of each element comes after.

.each_value{|x| do something to x} This method of enumeration loops through each value of a hash, where x is the value, and then what you want to perform on each value.

.each_key{|y| do something to y} This method of enumeration loops through each key of a hash, where y is the key, and then what you want to perform on each value.

.each_index {|x| do something to x} This method of enumeration loops through each index of an array, where x is the index, and then what you want to perform on each index.

.include?(x) This method of enumeration checks to see if an array or has includes a value (x).

.collect(function) or .reduce(function) These methods of enumeration reduce all the elements of an array to a single value. For example: [1,2,3,4,5].reduce(:+), will return 15 by getting the sum of all the elements of the array.
